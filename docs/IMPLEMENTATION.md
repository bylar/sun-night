# 谋定三国天下 · 事务安排甘特图工具

## 项目概述

一款专为《谋定三国天下》手游设计的 **H5 竖列甘特图** 事务安排辅助工具，帮助玩家直观管理每日游戏任务、活动时间、资源采集等事务。

---

## 技术栈

| 类别 | 技术 | 版本 |
|------|------|------|

> - 丰富的表单、标签、弹窗等组件
> - 社区活跃，文档完善

## 核心设计

### 数据模型

```typescript
interface TaskItem {
  id: string           // 唯一标识
  name: string         // 任务名称（如"开交宣战"、"纳粮"）
  category: TaskCategory // 类别枚举
  priority: TaskPriority  // 高/中/低优先级
  startTime: string    // "HH:mm"
  endTime?: string     // 可选，持续型任务
  description?: string // 详细说明
  isHighlighted?: boolean // 是否高亮（重点事项）
  color?: string       // 自定义背景色
  count?: number       // 右侧计数器值
  icon?: string        // Vant图标名
}
```

### 甘特图渲染逻辑

```
┌─────────────────────────────────────┐
│ NavBar (渐变紫)                      │ ← 固定顶部导航

│ [上一天] 第17天(三) [下一天]          │ ← 日期切换
├─────────────────────────────────────┤
│ ●建筑 ●军事 ●资源 ●活动 ●日常 ●特殊    │ ← 图例筛选
├─────────────────────────────────────┤
│ ┌────┬────────────────┬────┐        │
│ │时间 │ 事务内容       │ #  │        │ ← 表头
│ ├────┼────────────────┼────┤        │
│ │12:00│[建]            │    │        │
│ │     │[开交战服大赏]★  │ 0  │        │ ← 绝对定位的任务行
│ │12:30│[宝]💎          │    │        │   top = (开始时间-起始) / 总时长 * 100%
│ │13:00│[开交宣战]★     │    │        │
│ │     ...             │    │        │
│ └────┴────────────────┴────┘        │
├─────────────────────────────────────┤
│ 📋共35项  🔥重点12项                 │ ← 固定底部统计
└─────────────────────────────────────┘
```
